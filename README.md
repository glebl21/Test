# 🎙️ Telegram Voice Transcriber Bot

Бот транскрибирует голосовые сообщения, кружки и аудиофайлы в текст.  
Использует только **бесплатные API** — без оплаты и карты.

---

## ✨ Возможности

| Функция | Описание |
|---|---|
| 🎤 Голосовые сообщения | Транскрибация ГС из Telegram |
| ⭕ Кружки (video note) | Транскрибация видео-кружков |
| 🎵 Аудиофайлы | MP3, OGG, WAV, FLAC, M4A, MP4 |
| 🌍 Выбор языка | Авто + RU, EN, DE, FR, ES, UK |
| 📝 Саммари | Краткое изложение одной кнопкой |
| 🔁 Пересказ | Кратко / Подробно / По пунктам |
| 🌐 Перевод | На 7 языков одной кнопкой |
| 📊 Статистика | Сколько ГС обработано |

---

## 🔑 Получение бесплатных ключей

### 1. Telegram Bot Token
1. Открой Telegram → найди **@BotFather**
2. Напиши `/newbot` → придумай имя и username
3. Скопируй токен

### 2. Groq API Key — транскрибация (Whisper)
1. Зайди на **https://console.groq.com**
2. Зарегистрируйся → **API Keys → Create API Key**
3. Скопируй ключ

**Бесплатные лимиты Groq:**
- Whisper: 7 200 секунд аудио / час

### 3. Gemini API Key — саммари и перевод
1. Зайди на **https://aistudio.google.com**
2. Нажми **Get API Key → Create API Key**
3. Скопируй ключ

**Бесплатные лимиты Gemini:**
- Gemini 2.0 Flash: 1 500 запросов / день

## 🚀 Локальный запуск

### Установка зависимостей
```bash
pip install pyTelegramBotAPI groq requests
```

### Вставь ключи в файл `voice_transcriber_bot.py`
```python
TELEGRAM_BOT_TOKEN = os.environ.get("TELEGRAM_BOT_TOKEN", "вставь_сюда")
GROQ_API_KEY = os.environ.get("GROQ_API_KEY", "вставь_сюда")
GEMINI_API_KEY = os.environ.get("GEMINI_API_KEY", "вставь_сюда")
```

### Запуск
```bash
python voice_transcriber_bot.py
```

## ☁️ Деплой на Railway (24/7 бесплатно)

### 1. Подготовь файлы
Убедись что у тебя есть три файла:
- `voice_transcriber_bot.py`
- `requirements.txt`
- `Procfile`

**requirements.txt:**
```
pyTelegramBotAPI
groq
requests
```

**Procfile:**
```
worker: python voice_transcriber_bot.py
```

### 2. Залей на GitHub
1. Зарегистрируйся на **https://github.com**
2. Создай новый репозиторий → **New repository**
3. Загрузи все три файла

> ⚠️ Не вставляй ключи в код перед загрузкой на GitHub — GitHub их обнаружит и заблокирует!

### 3. Задеплой на Railway
1. Зайди на **https://railway.app** → войди через GitHub
2. **New Project → Deploy from GitHub repo**
3. Выбери свой репозиторий
4. Перейди в **Variables** и добавь:

| Имя переменной | Значение |
|---|---|
| `TELEGRAM_BOT_TOKEN` | твой токен от @BotFather |
| `GROQ_API_KEY` | твой ключ от console.groq.com |
| `GEMINI_API_KEY` | твой ключ от aistudio.google.com |

5. Railway автоматически задеплоит бота

## 📋 Как пользоваться

1. Отправь боту **голосовое сообщение**, **кружок** или **аудиофайл**
2. Бот пришлёт **транскрипцию** с тремя кнопками:
   - **📝 Саммари** — краткое изложение
   - **🔁 Пересказ** — выбери стиль (кратко / подробно / по пунктам)
   - **🌐 Перевести** — выбери язык перевода

---

## ⚠️ Ограничения

- Максимальный размер файла: **20 МБ** (ограничение Telegram)
- Бот не отвечает на обычные текстовые сообщения в группе
- После перезапуска бота кнопки старых сообщений не работают — нужно отправить голосовое заново
